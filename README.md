# i3config
My i3 config files. Made for use with arc theme.

# Dependencies
Required:
- i3-gaps (or i3, if you comment out some lines in config)
- rofi
- i3blocks  
- fonts-font-awesome

Not quite required:
- compton
- xautolock
- feh


# installation
clone into ~.config/, delete ~.config/i3/, rename i3config to i3
